CryptoTrackr

Project Description
This project is a simple Crypto Dashboard that shows real-time cryptocurrency data. It uses the CoinGecko API to fetch information like price, market cap, and daily changes of different cryptocurrencies.

The dashboard displays the data in a clean and user-friendly way so users can easily track crypto market trends.


API Used
CoinGecko API (free and no API key required)

Features
Shows live cryptocurrency prices
Displays market data (market cap, volume, 24h change)
Search for different cryptocurrencies
Simple and responsive design

Technologies Used
HTML
CSS
JavaScript
CoinGecko API
